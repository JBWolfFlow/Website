# Reporting Security Vulnerabilities
#
# If you discover a security vulnerability, please report it to us at:
# security@ayra-tech.net
#
# Please include:
# - Description of the vulnerability
# - Steps to reproduce
# - Potential impact
# - Any proof-of-concept code
#
# We will acknowledge receipt within 24 hours and provide a detailed
# response within 72 hours.
#
# Please do not publicly disclose the vulnerability until we have had
# a chance to address it.
#
# Thank you for helping keep AryaTech secure!
